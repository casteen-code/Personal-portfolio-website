# Casper 个人项目文件夹

这个文件夹现在按用途分成 5 类，打开后可以直接按名字找东西。

## 文件夹说明

- `01-个人知识库`: 存放你的职业经历、项目案例、能力证据、岗位定制需求等长期素材。
- `02-网页简历`: 你的个人简历网页，双击 `index.html` 可以本地预览。
- `03-简历生成工具`: 生成 Word 简历的脚本工具。
- `04-简历输出`: 已生成的 Word 简历等成品文件。
- `99-项目记录`: 对话恢复记录、过程记录、临时说明等归档内容。

## 推荐使用方式

1. 新增职业素材时，先放进 `01-个人知识库/00-快速投喂区.md`。
2. 要更新网页简历时，主要修改 `02-网页简历`。
3. 要生成 Word 简历时，使用 `03-简历生成工具/build_general_resume.py`。
4. 生成后的简历统一放在 `04-简历输出`。
5. 不确定放哪里时，先放进 `99-项目记录`，后续再整理。
