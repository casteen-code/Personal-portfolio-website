# Casper 网页简历

这是一个适合 GitHub Pages 发布的个人简历网页。

## 文件说明

- `index.html`: 简历内容和页面结构
- `style.css`: 页面视觉样式
- `script.js`: 中英文切换
- `assets/profile.jpg`: 头像照片

## 本地预览

直接双击打开 `index.html` 即可预览。

## GitHub Pages 发布

1. 在 GitHub 创建一个新仓库，例如 `resume`。
2. 上传本文件夹内的所有文件。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，文件夹选择 `/root`。
6. 保存后等待 GitHub 生成网页链接。
